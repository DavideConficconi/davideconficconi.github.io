TiReX

TiReX (a.k.a. Tiled Regular eXpressions matching architecture) is a Domain Specific Architecture for the Regular Expressions (REs) matching tasks. This work has been part of my, and my colleague and friend A. Comodi, Master Thesis in Computer Science and Engineering. Moreover with that project we have been finalist in the Xilinx Open HardWare competition in 2017. That version is open source and available at this Bitbucket repository.

SDAccel-Chisel Integration

I collaborated with L. Di Tucci in the creation of a Chisel Wrapper for SDAccel RTL Flow, which is open source at this github repository. By means of this wrapper, every Chisel developer can emulate a kernel in a complete integrated way

BISMO

During my research stay at Xilinx in Dublin, I collaborated with the team of the BISMO project. BISMO is a programmable FPGA accelerator for few-bit integer matrix multiplication.

IRON

IRON is a framework, developed with with E. D'Arnese, which enables both Image Registration and Mutual Information computations to FPGA acceleration. It is open sourced at this github repository, and it was a finalist of the Xilinx Open HardWare design contest 2019.
