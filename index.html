TiReX

TiReX (a.k.a. Tiled Regular eXpressions matching architecture) is a Domain Specific Architecture for the Regular Expressions (REs) matching tasks. This work has been part of my, and my colleague and friend A. Comodi, Master Thesis in Computer Science and Engineering. Moreover with that project we have been finalist in the Xilinx Open HardWare competition in 2017. That version is open source and available at this Bitbucket repository. This work leads to a publication: "TiReX: Tiled Regular eXpression matching architecture." A. Comodi, D. Conficconi, A. Scolari, M. D. Santambrogio 2018 IEEE International Parallel and Distributed Processing Symposium Workshops (IPDPSW). IEEE, 2018. p. 131-137.

SDAccel-Chisel Integration

I collaborated with L. Di Tucci in the creation of a Chisel Wrapper for SDAccel RTL Flow, which is open source at this github repository. By means of this wrapper, every Chisel developer can emulate a kernel in a complete integrated way. This work is also part of a publication: "A Parallel, Energy Efficient Hardware Architecture for the merAligner on FPGA using Chisel HCL." L. Di Tucci, D.Conficconi, A. Comodi, S. Hofmeyr, D. Donofrio, M. D. Santambrogio 2018 IEEE International Parallel and Distributed Processing Symposium Workshops (IPDPSW). IEEE, 2018. p. 214-217.

BISMO

During my research stay at Xilinx in Dublin, I collaborated with the team of the BISMO project. BISMO is a programmable FPGA accelerator for few-bit integer matrix multiplication. This work leads to a publication: "Optimizing Bit-Serial Matrix Multiplication for Reconfigurable Computing." Y. Umuroglu, D.Conficconi, L. Rasnayake, T. Preusser, M. Själander 2019 ACM Transactions on Reconfigurable Technology and Systems (TRETS) 12.3 (2019): 1-24.

IRON

IRON is a framework, developed with E. D'Arnese, which enables both Image Registration and Mutual Information computations to FPGA acceleration. It is open sourced at this github repository, and it was a finalist of the Xilinx Open HardWare design contest 2019.

A Framework for Customizable FPGA-based Image Registration Accelerators

Starting from IRON, along with E. D'Arnese and E. Del Sozzo, we developed a framework for generating customizable FPGA-based accelerators of the MI computations, particularly devoted at Image Registration computation. Our paper has been accepted and published at the 2021 ACM/SIGDA International Symposium on Field-Programmable Gate Arrays (FPGA'21) and it is open-sourced at this github repository.

Xilinx Project Template

Together with E. Del Sozzo, we are developing a template repository for Xilinx-FPGA-based projects that starts from IRON as an example project. Our template provide an example of code organization, flows, utilities and makefiles for both Zynq-based (both SoC and MPSoC) devices and PCIe accelerator cards (e.g., Alveo cards), exploiting etiher PYNQ-based APIs or C/C++/OpenCL APIs. Currently focus on HLS-based, but we are working to extend for multi-code-base and provide some examples for RTL flows (IP packaging for both Vivado and Vitis flow). It is open-sourced at this github repository.
